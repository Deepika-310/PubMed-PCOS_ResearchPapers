# PubMed-PCOS_ResearchPapers
This project aims to collect data regarding the research papers published on the topic (sample topic: PCOS in sedantary lifestyle) from pubmed website with the help of web scraping
Developed an automated Python pipeline using Requests, BeautifulSoup, and Pandas to scrape and organize PubMed research articles on PCOS and sedentary lifestyles, cleaning and structuring the dataset and exporting it to Excel for analysis.
